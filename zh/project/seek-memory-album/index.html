寻忆

Apr 17, 2019
视频

目前，AI技术高速发展，语音、图像、自然语言处理等领域成果不断。然而，AI算法落地成为众多新兴企业深入思考的关键问题。基于图像识别的算法落地已在安防、门禁、工业检测、自动驾驶等领域有了产业化的突破，而AI的潜在落地应用场景远不止这些。基于图像和视频指纹技术，我们构建了智能云相册平台，旨在成为用户快速精准寻找相册文件的好帮手。

考虑到相册的主要和特色功能，考虑到简洁性和易用性，我们将平台命名为“寻忆”，意在寻找回忆，品牌标语即为“没有难以寻找的回忆”。我们将平台标志设计为树叶，从形态上看，也可以理解为羽毛制作的钢笔，包含“记录”之意，从另一个角度来讲，是树叶制作而成的书签，在相册中起到帮助主人寻找所需的作用。网页主题使用绿色和橙色相间，即为四季落叶之色，营造唯美意境。
